# 09-单据命名规则

| 单据 | 前缀 | 格式 | 示例 |
|:--|:--|:--|:--|
| 收货单 | RCV | RCV{YYYYMMDD}-{4位序号} | RCV20260705-0001 |
| 上架单 | PUT | PUT{YYYYMMDD}-{4位序号} | PUT20260705-0001 |
| 波次单 | WAVE | WAVE{YYYYMMDD}-{4位序号} | WAVE20260705-0001 |
| 拣货单 | PICK | PICK{YYYYMMDD}-{4位序号} | PICK20260705-0001 |
| 复核单 | CHECK | CHECK{YYYYMMDD}-{4位序号} | CHECK20260705-0001 |
| 包裹 | PKG | PKG{YYYYMMDD}-{6位序号} | PKG20260705-000001 |
| 交运单 | DSH | DSH{YYYYMMDD}-{4位序号} | DSH20260705-0001 |
| 盘点单 | CK | CK{YYYYMMDD}-{4位序号} | CK20260705-0001 |
| 调拨单 | TR | TR{YYYYMMDD}-{4位序号} | TR20260705-0001 |
| 报损单 | BL | BL{YYYYMMDD}-{4位序号} | BL20260705-0001 |
| 库存流水 | FL | FL{YYYYMMDD}-{8位序号} | FL20260705-00000001 |

> 序号每日从 0001 起递增。已确认/已作废的单号不回收。
