# 08-调拨流程

> 强盛科技二期内容。仓间调拨 + 与进销存 ERP 调拨单对接。

## 调拨类型

| 类型 | 说明 |
|:--|:--|
| 仓间调拨 | WMS 内两个仓库之间 |
| ERP 调拨 | 进销存 ERP 下发调拨单到 WMS 执行 |

## 流程（分步调拨）

```
ERP/WMS创建调拨单(TR) → 调出库确认出库(扣现存,改在途)
→ 调入库收货清点 → 实收=调出? → 是:确认入库(加现存,消在途)
                                → 否:按实收入库+生成报损单(BL)记录差异
```

## 业务规则
- 调拨在途库存不可销售（状态=在途）
- 到货差异自动生成报损单，原因标记"调拨损耗"
- 调入仓库仅按实收数量入库
