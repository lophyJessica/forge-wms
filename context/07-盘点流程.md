# 07-盘点流程

## 盘点类型

| 类型 | 说明 |
|:--|:--|
| 明盘 | 显示系统库存数，仓管逐项清点确认 |
| 盲盘 | 不显示系统数，纯实物清点后对比差异 |
| 动盘 | 不停业盘点，仅锁定当前扫描的货位 |
| 全盘 | 全仓停业盘点，全部货位锁定 |

## 流程

```
创建盘点单(CK) → 锁定盘点范围 → PDA扫描货位+商品+实盘数
→ 系统比对差异 → 差异报告 → 主管审核 → 库存调整 → 生成FL
```

## 业务规则
- 盘点中对应货位冻结，禁止出入库
- 差异超过阈值(>5%) → 主管重盘
- 确认调整后自动更新现存 + 生成盘点流水
