# 03-主要功能模块和清单

> 强盛科技 WMS，单一仓库单货主模式，PC + PDA 双端。

## 一期功能模块（6大模块）

### 1. 基础数据管理
- 仓库档案 + 库区管理（收货区/存储区/拣货区/退货区/发货区）
- 货位管理：货架编号/层/列，支持条码绑定
- 商品/供应商档案：从进销存 ERP 同步

### 2. 入库管理
- **收货**：PC 创建收货单，关联采购订单 PO，打印收货标签
- **质检**：抽检/全检，合格入库/不合格退货
- **上架**：PDA 扫描货位条码，系统推荐空闲货位或人工指定
- 单据：收货单(RCV) → 上架单(PUT)

### 3. 出库管理（日均2万+单）
- **波次**：系统波次（按承运商/线路/优先级聚合）+ 手动波次
- **拣货**：PDA 按波次领任务 → 扫描货位 → 扫描商品 → 确认
- **播种**：多订单商品按订单分播到对应发货箱
- **复核**：扫描发货箱/商品，校验数量
- **包装**：称重、贴面单、生成包裹号
- **交运**：交接承运商，回传 TMS
- 单据：波次(WAVE) → 拣货(PICK) → 复核(CHECK) → 包裹(PKG) → 交运(DSH)

### 4. 库存管理
- 库存状态：可用/占用/冻结/在途
- 库存查询：按仓库/库区/货位/商品/批次多维度
- 库存流水：自动记录所有变动
- 库存预警：低于安全库存标黄，低于最低标红

### 5. 盘点管理
- 明盘 + 盲盘
- 盘点单(CK) → 差异报告 → 主管审核 → 库存调整

### 6. PDA 端
- 收货上架/拣货出库/盘点
- 离线模式：无网缓存，恢复后自动同步

## 全国部署
6 个仓库，出库 20,000+ 单/天
