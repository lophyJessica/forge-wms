# 11-字段校验规范

| 字段类型 | 规则 |
|:--|:--|
| 必填字段 | 提交时校验，空值标红提示 |
| 数量 | 正整数，>0，≤上限值 |
| 金额 | ≥0，2位小数 |
| 日期 | 不超当前日期（补录场景除外） |
| 编码 | 字母+数字，不可重复 |
| 备注 | 上限 200 字符 |
| 条码 | 数字为主，支持 EAN-13 |
| 单号 | 系统自动生成，不可编辑 |

## 校验时机

| 时机 | 校验范围 |
|:--|:--|
| 实时（失焦） | 格式/范围 |
| 保存草稿 | 宽松校验（仅严重错误阻断） |
| 确认/提交 | 全量严格校验 |
