# 商品档案_前端Demo版PRD

> 版本：V1.0 | 更新时间：2026-07-07
> 读者：前端 Demo 实现、测试、产品复核
> 文档定位：定义商品档案 Demo 的列表页、新增编辑页、详情页页面结构、控件、交互与 Mock 数据。字段定义、枚举、必填性和校验规则统一引用《商品档案字段清单》，本文不重复定义。

## 1. 页面清单

| 页面 | 建议路由 | 页面目标 |
| :--- | :--- | :--- |
| 商品列表页 | `/base/products` | 查询商品、查看 ERP 基础信息和 WMS 仓储属性、进入新增 / 编辑 / 详情、触发启停用 |
| 新增商品仓储属性页 | `/base/products/new` | 从 ERP 已同步商品中选择一条，补充 WMS 仓储属性 |
| 编辑商品仓储属性页 | `/base/products/:code/edit` | 查看 ERP 只读字段，维护 WMS 仓储属性 |
| 商品详情页 | `/base/products/:code` | 只读查看商品资料、仓储属性、系统信息与引用说明 |

## 2. Demo 数据口径

| 项目 | 规则 |
| :--- | :--- |
| 数据来源 | Dexie.js `products` 表模拟 ERP 同步商品 + WMS 仓储属性 |
| ERP 字段 | 商品编码、名称、规格、单位、条码、分类、税率等只读展示 |
| WMS 字段 | 安全库存、最低库存、默认货位、体积、重量、是否批次管理、备注可维护 |
| 状态 | `status` 仅通过列表页 / 详情页“启用 / 停用”按钮变更 |
| 删除 | 不提供删除按钮、删除菜单或物理删除交互 |
| 日期 | Mock 日期统一使用 2026 年 |

## 3. 列表页

### 3.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 标题区 | 标题“商品档案”、新增商品仓储属性按钮 | 新增按钮进入新增页，用于维护已同步商品的 WMS 属性 |
| 查询区 | 综合搜索、商品分类、状态、是否批次管理、查询按钮、重置按钮 | 枚举和字段口径引用字段清单 |
| 表格区 | 商品列表、仓储阈值、状态标识、行操作 | 默认分页 20 条；空数据展示“暂无商品档案” |

### 3.2 查询控件

| 控件 | 类型 | 交互 |
| :--- | :--- | :--- |
| 综合搜索 | Input | 支持商品编码 / 商品名称 / 商品条码模糊查询；回车或点击查询触发 |
| 商品分类 | Select | 默认全部；选项来源于 ERP 同步分类 |
| 状态 | Select | 默认全部；包含启用、停用 |
| 是否批次管理 | Select | 默认全部；包含是、否 |
| 查询 | Button | 触发筛选 |
| 重置 | Button | 清空条件并刷新列表 |

### 3.3 表格列与行操作

| 列 | 展示规则 |
| :--- | :--- |
| 商品编码 | 等宽文本，点击可进入详情页 |
| 商品名称 | 普通文本，长文本截断 |
| 规格型号 | 普通文本，空值展示 `-` |
| 计量单位 | 普通文本 |
| 商品条码 | 等宽文本，空值展示 `-` |
| 商品分类 | 普通文本，空值展示 `-` |
| 安全库存 | 数字右对齐，空值展示 `-` |
| 最低库存 | 数字右对齐，空值展示 `-` |
| 默认货位 | 等宽文本，空值展示 `-` |
| 是否批次管理 | 是 / 否 |
| 状态 | StatusPill，启用 / 停用使用不同颜色 |
| 操作 | 查看、编辑、停用 / 启用 |

行操作规则：

- 查看：进入商品详情页。
- 编辑：进入编辑商品仓储属性页。
- 停用：仅 `ENABLED` 行展示，点击后弹出确认框，确认后状态变为 `DISABLED`。
- 启用：仅 `DISABLED` 行展示，点击后弹出确认框，确认后状态变为 `ENABLED`。
- 不提供删除入口。
- 按通用规范，当前状态下不可用的动作隐藏，不展示灰色 disabled 态。

## 4. 新增 / 编辑页

### 4.1 页面结构

| 区域 | 新增模式 | 编辑模式 |
| :--- | :--- | :--- |
| 顶部导航 | 返回按钮、标题“新增商品仓储属性” | 返回按钮、标题“编辑商品 {code}” |
| ERP 同步信息 | 先选择 ERP 已同步商品；选择后展示只读字段 | 展示商品编码、名称、规格、单位、条码、分类、税率等只读字段 |
| WMS 仓储属性 | 维护安全库存、最低库存、默认货位、体积、重量、是否批次管理、备注 | 同新增模式，可编辑 |
| 操作区 | 取消、保存 | 取消、保存 |

### 4.2 字段编辑权限

| 字段组 | 字段 | 新增模式 | 编辑模式 |
| :--- | :--- | :--- | :--- |
| ERP 同步信息 | 商品编码、商品名称、规格型号、计量单位、商品条码、商品分类、税率、参考采购价 | 选择商品后只读展示 | 只读展示 |
| WMS 仓储属性 | 安全库存、最低库存 | 可编辑 | 可编辑 |
| WMS 仓储属性 | 默认货位 | 从启用货位档案中选择 | 从启用货位档案中选择 |
| WMS 仓储属性 | 体积、重量 | 可编辑 | 可编辑 |
| WMS 仓储属性 | 是否批次管理 | 可编辑，默认否 | 可编辑 |
| WMS 仓储属性 | 备注 | 可编辑 | 可编辑 |
| 状态 | `status` | 不展示 | 不展示 |

状态字段不在新增 / 编辑主表单中展示；状态只能在列表页或详情页通过启用 / 停用按钮变更。

### 4.3 新增模式特殊规则

| 场景 | 交互 |
| :--- | :--- |
| 选择 ERP 商品 | 使用 SelectWithSearch，支持商品编码 / 名称 / 条码搜索 |
| 商品已维护 WMS 属性 | 不进入可选列表，避免重复维护同一商品档案 |
| 商品未同步 | 不允许手工录入商品编码创建；提示“请先从 ERP 同步商品” |
| 选择后展示 | ERP 字段自动带出并只读；光标默认进入安全库存字段 |

### 4.4 保存交互

| 场景 | 交互 |
| :--- | :--- |
| 新增成功 | Toast 提示“商品仓储属性已新增”；返回列表页并展示新记录 |
| 编辑成功 | Toast 提示“商品仓储属性已保存”；返回列表页并刷新数据 |
| 保存失败 | 阻断提交，定位首个错误字段或展示错误提示 |
| 阈值关系错误 | 提示“最低库存不能大于安全库存” |
| 默认货位无效 | 提示“默认货位不存在或已停用” |
| 取消 | 返回列表页，不保存当前输入 |
| 编辑不存在编码 | Toast / Alert 提示“商品档案不存在”，返回列表页 |

## 5. 详情页

### 5.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 顶部区 | 返回按钮、标题、状态标识、编辑按钮、启用 / 停用按钮 | 启停用按钮按当前状态切换展示 |
| ERP 同步信息 | 商品编码、名称、规格、单位、条码、分类、税率、参考采购价、ERP 更新时间 | 只读展示，空值展示 `-` |
| WMS 仓储属性 | 安全库存、最低库存、默认货位、体积、重量、是否批次管理、备注 | 只读展示，编辑需进入编辑页 |
| 引用说明 | 商品快照说明、库存预警阈值来源说明 | 展示为只读说明区域，不提供操作 |
| 系统信息 | WMS 建档时间、最后修改时间 | 使用 `YYYY-MM-DD HH:mm:ss` |

### 5.2 详情页交互

- 点击编辑进入编辑商品仓储属性页。
- 点击停用 / 启用弹出二次确认，确认后刷新详情页状态。
- 停用后详情页仍可查看，ERP 同步字段和 WMS 仓储属性不隐藏。
- 不提供删除入口。

## 7. 验收清单

| 验收ID | 验收项 | 标准 |
| :--- | :--- | :--- |
| FE-AC01 | 列表查询 | 综合搜索、分类、状态、是否批次管理可组合筛选，重置后恢复全部数据 |
| FE-AC02 | ERP 字段只读 | 新增和编辑页中 ERP 同步字段均不可编辑，只能选择或查看 |
| FE-AC03 | 新增维护 | 只能从 ERP 已同步商品中选择后补充 WMS 仓储属性，不允许手工创建商品编码 |
| FE-AC04 | 编辑保存 | 编辑页可修改安全库存、最低库存、默认货位、体积、重量、是否批次管理、备注，保存后列表和详情同步更新 |
| FE-AC05 | 阈值校验 | 最低库存大于安全库存时阻断保存并提示 |
| FE-AC06 | 默认货位校验 | 只能选择启用货位；停用货位不进入候选 |
| FE-AC07 | 状态变更 | 启用 / 停用通过按钮触发并二次确认，确认后列表与详情状态同步刷新 |
| FE-AC08 | 删除禁用 | 页面不出现删除按钮、删除菜单或物理删除交互 |
| FE-AC09 | 详情展示 | 详情页分组展示 ERP 同步信息、WMS 仓储属性、引用说明、系统信息 |
| FE-AC10 | 快照提示 | 页面说明商品档案变更不影响已有单据商品快照 |
| FE-AC11 | Mock 日期 | Mock 数据中的系统时间使用 2026 年 |
