# 商品档案字段清单

> 版本：V1.0 | 更新时间：2026-07-07
> 文档定位：商品档案字段的 SSOT。主PRD、前端 Demo PRD、测试用例、库存预警和业务单据只引用本清单字段口径，不重复定义枚举、必填性和校验规则。
> 适用范围：WMS 基础数据管理 / 商品档案。

---

## 1. SSOT 原则

- 商品基础信息来源于进销存 ERP，同步到 WMS 后只读展示。
- WMS 只维护仓储属性，不反向修改 ERP 商品编码、名称、规格、单位、条码、分类、税率、价格等字段。
- 下游业务单据保存商品快照；商品档案后续变更不影响已有单据。
- 库存预警只读引用 `safetyStock`、`minimumStock`，不在预警页补录阈值。
- 状态变更只能通过“启用 / 停用”动作按钮触发，不允许在新增 / 编辑表单中直接修改 `status`。

## 2. ERP 同步字段（只读）

| 字段名称 | Field | 来源 | 类型 | 必填 | 枚举 / 取值 | 校验规则 | 页面使用 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 商品编码 | `code` | ERP 同步 | String | 是 | - | ERP 保证唯一；WMS 按 `code` 唯一存储；格式以 ERP 商品档案为准，当前 Demo 使用 `SKU001` 口径 | 列表、详情、新增只读、编辑只读、候选展示 | 商品主键；下游单据快照字段通常命名为 `productCode` |
| 商品名称 | `name` | ERP 同步 | String | 是 | - | 非空；去首尾空格；建议 1-50 字符；WMS 不允许编辑 | 列表、详情、新增只读、编辑只读、综合搜索 | 商品标准名称 |
| 规格型号 | `productSpec` | ERP 同步 | String | 是 | - | 非空；建议 1-100 字符；WMS 不允许编辑 | 列表、详情、新增只读、编辑只读 | 区分同品类不同规格 |
| 计量单位 | `unit` | ERP 同步 | String / Enum | 是 | ERP 枚举未在 context 明确；Demo 使用 `盒`、`支`、`包`、`台`、`个` | 必须有值；WMS 按同步值展示，不维护单位换算 | 列表、详情、新增只读、编辑只读 | 入库、出库、库存数量的展示单位 |
| 商品条码 | `barcode` | ERP 同步 | String | 否 | 数字为主，支持 EAN-13 | 填写时需符合条码规则；同一启用商品条码不得重复；WMS 不允许编辑 | 列表、详情、新增只读、编辑只读、综合搜索、PDA 扫码 | PDA 收货、上架、拣货、复核扫码校验使用 |
| 商品分类 | `category` | ERP 同步 | String / Enum | 不确定 | 分类枚举未在 context 明确 | WMS 只读展示；分类值为空时展示 `-` | 列表、详情、筛选、新增只读、编辑只读 | 用于查询筛选和运营分类展示 |
| 税率 | `taxRate` | ERP 同步 | Decimal | 不确定 | 取值范围以 ERP 为准 | WMS 只读展示；展示前校验为非负数；格式建议百分比，保留 2 位小数；异常时标记同步数据异常 | 详情、新增只读、编辑只读 | context 与当前 Demo 未定义税率，按用户要求列入 ERP 同步字段 |
| 参考采购价 | `defaultPurchasePrice` | ERP 同步 | Decimal | 否 | - | 金额 `>=0`，2 位小数；WMS 只读展示，不作为 WMS 库存金额口径 | 详情、新增只读、编辑只读 | 当前 WMS Demo 已有该字段；正式字段名需与 ERP 接口确认 |
| ERP 更新时间 | `erpUpdatedAt` | ERP 同步 | DateTime | 否 | `YYYY-MM-DD HH:mm:ss` | WMS 只读展示；格式异常时展示 `-` | 详情 | 用于判断基础信息同步新鲜度 |

## 3. WMS 仓储属性（WMS 维护）

| 字段名称 | Field | 来源 | 类型 | 必填 | 枚举 / 取值 | 校验规则 | 页面使用 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 安全库存 | `safetyStock` | WMS 维护 | Integer | 否，建议维护 | - | 非负整数；填写时 `>=0`；若同时填写最低库存，必须满足 `minimumStock <= safetyStock` | 列表、详情、新增、编辑、库存预警引用 | 库存预警标黄阈值；未维护时预警页展示 `-` |
| 最低库存 | `minimumStock` | WMS 维护 | Integer | 否，建议维护 | - | 非负整数；填写时 `>=0`；若同时填写安全库存，必须满足 `minimumStock <= safetyStock` | 列表、详情、新增、编辑、库存预警引用 | 库存预警标红阈值；必须在商品档案落地 |
| 默认货位 | `defaultLocationCode` | WMS 维护 / 货位档案 | String | 否 | 来源于货位档案 | 必须选择启用货位；不得手工输入游离值；停用货位不可被新设为默认货位 | 列表、详情、新增、编辑 | 上架推荐参考，不替代 PDA 实扫货位 |
| 体积 | `volume` | WMS 维护 | Decimal | 否 | 单位未明确 | 填写时必须 `>=0`；建议最多 3 位小数；单位需在页面标题或字段后缀展示 | 详情、新增、编辑 | 用于仓储作业参考；单位 context 未明确 |
| 重量 | `weight` | WMS 维护 | Decimal | 否 | 单位未明确 | 填写时必须 `>=0`；建议最多 3 位小数；单位需在页面标题或字段后缀展示 | 列表可选、详情、新增、编辑 | 可供包装、称重或库内作业参考；单位 context 未明确 |
| 是否批次管理 | `batchManaged` | WMS 维护 | Boolean | 是 | `true`、`false` | 新增默认 `false`；启用后新入库 / 库存记录需记录批次；历史单据不回写 | 列表、详情、新增、编辑、筛选 | 控制商品是否按批次维度管理库存 |
| 状态 | `status` | WMS 系统生成 / 动作按钮变更 | Enum | 系统自动 | `ENABLED`、`DISABLED` | 新增默认 `ENABLED`；仅允许通过启用 / 停用按钮变更；不在表单中提供下拉 | 列表、详情、筛选、候选过滤 | 控制商品能否进入新业务候选范围 |
| 备注 | `remark` | WMS 维护 | Text | 否 | - | 上限 200 字符；不作为系统判断条件 | 详情、新增、编辑 | 仓储侧补充说明 |
| WMS 建档时间 | `createdAt` | WMS 系统生成 | DateTime | 系统自动 | `YYYY-MM-DD HH:mm:ss` | 新增维护保存时写入；不允许手工编辑 | 详情、Mock 数据 | 商品进入 WMS 商品档案的时间 |
| 最后修改时间 | `updatedAt` | WMS 系统生成 | DateTime | 系统自动 | `YYYY-MM-DD HH:mm:ss` | 编辑仓储属性或状态变更后写入；不允许手工编辑 | 列表、详情、Mock 数据 | 最近一次 WMS 侧修改时间 |

## 4. 枚举定义

### 4.1 状态 `BaseDataStatus`

| 枚举值 | 展示文案 | 是否可被新业务引用 | 说明 |
| :--- | :--- | :---: | :--- |
| `ENABLED` | 启用 | 是 | 可被新建收货、上架、拣货、盘点、调拨、PDA 作业选择 |
| `DISABLED` | 停用 | 否 | 不可被新业务选择，但历史单据、库存查询、库存流水仍可展示 |

### 4.2 是否批次管理 `batchManaged`

| 值 | 展示文案 | 说明 |
| :--- | :--- | :--- |
| `true` | 是 | 新入库、库存查询按批次维度记录；批次规则以库存模块为准 |
| `false` | 否 | 不强制维护批次；库存批次展示可为 `-` |

## 5. 校验规则

| 校验对象 | 规则 | 触发时机 | 失败提示 |
| :--- | :--- | :--- | :--- |
| ERP 同步字段编辑 | `code`、`name`、`productSpec`、`unit`、`barcode`、`category`、`taxRate` 等只读 | 页面渲染 | 不提供可编辑入口 |
| 商品编码唯一 | `code` 在商品档案内全局唯一 | ERP 同步入库 / Demo 新增保存 | 商品编码已存在 |
| 条码格式 | 数字为主，支持 EAN-13；填写时不可与其他启用商品重复 | ERP 同步校验 / PDA 扫码 | 商品条码格式异常或已存在 |
| 安全库存 | 非负整数 | 失焦、保存 | 安全库存必须为非负整数 |
| 最低库存 | 非负整数 | 失焦、保存 | 最低库存必须为非负整数 |
| 库存阈值关系 | 同时填写时 `minimumStock <= safetyStock` | 保存 | 最低库存不能大于安全库存 |
| 默认货位 | 必须来源于启用货位档案 | 选择、保存 | 默认货位不存在或已停用 |
| 体积 / 重量 | 填写时必须为非负数 | 失焦、保存 | 体积/重量不能小于 0 |
| 批次管理 | 必须为布尔值 | 保存 | 是否批次管理取值异常 |
| 状态变更 | `status` 只能由启用 / 停用动作按钮改变 | 点击动作按钮 | 不展示状态下拉 |
| 备注长度 | `remark` 不超过 200 字符 | 输入、保存 | 备注不能超过 200 字符 |

校验时机遵循《11-字段校验规范》：格式 / 范围可失焦校验，提交保存做全量严格校验；空值展示遵循通用规范使用 `-`。

## 6. 查询与展示规则

| 场景 | 字段 | 规则 |
| :--- | :--- | :--- |
| 综合搜索 | `code`、`name`、`barcode` | 一个 Input，支持商品编码、商品名称、商品条码模糊查询；PDA 场景扫码优先 |
| 分类筛选 | `category` | Select，选项来源于 ERP 同步分类；为空展示 `-` |
| 状态筛选 | `status` | Select，包含全部、启用、停用 |
| 批次管理筛选 | `batchManaged` | Select，包含全部、是、否 |
| 列表默认列 | `code`、`name`、`productSpec`、`unit`、`barcode`、`category`、`safetyStock`、`minimumStock`、`batchManaged`、`status`、`updatedAt` | 数量字段右对齐；空值展示 `-` |
| 候选展示 | `code`、`name`、`productSpec`、`unit` | 下拉候选建议展示为 `商品编码 商品名称 规格 单位` |
| 默认货位展示 | `defaultLocationCode` | 建议同步展示货位编码；如需要名称，来源于货位档案实时查询 |

## 7. 引用规则

| 引用方 | 引用字段 | 规则 |
| :--- | :--- | :--- |
| 收货单 RCV | `productCode`、`productName`、`productSpec`、`unit`、`productBarcode` | 继承 PO / 商品档案快照，单据生成后不随档案变更 |
| 上架单 PUT | 商品快照、`batchManaged`、`defaultLocationCode` | 商品快照继承 RCV；默认货位仅做推荐参考 |
| 拣货单 PICK | 商品快照、`barcode` | PDA 扫商品条码时校验商品归属和条码匹配 |
| 库存查询 | 商品快照、`batchManaged` | 支持按商品、批次查询；批次维度受 `batchManaged` 影响 |
| 库存预警 | `safetyStock`、`minimumStock` | 阈值来源固定为商品档案；预警页只读展示 |
| 盘点 / 调拨 | 商品快照、`status` | 新建作业只允许选择 `ENABLED` 商品；历史作业不受停用影响 |

## 8. 不确定性说明

| 事项 | 说明 |
| :--- | :--- |
| ERP 字段全集 | context 只明确商品/供应商从 ERP 同步，未列出完整接口字段；本文按用户要求和 ERP 商品档案参照物列出编码、名称、规格、单位、条码、分类、税率、参考采购价等字段。 |
| 税率字段 | 当前 WMS Demo 未实现 `taxRate`，context 未定义税率精度、必填性和展示位置；本文标注为 ERP 同步只读，正式以 ERP 接口为准。 |
| 单位枚举 | ERP 参照物有单位枚举，当前 WMS Demo 使用中文单位文本；本文不强行定义正式枚举。 |
| 安全 / 最低库存必填性 | context 未说明阈值是否必填；本文定义为选填但建议维护，库存预警在缺失时展示 `-`。 |
| 默认货位维度 | 多仓场景下默认货位可能需要按仓库维护；context 未明确，本文按商品级单默认货位处理，并标注需复核。 |
| 体积 / 重量单位 | context 未定义单位；本文只定义字段与非负校验。 |
