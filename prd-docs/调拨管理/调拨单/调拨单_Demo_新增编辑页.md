# 调拨单_Demo_新增编辑页

> 角色：Demo·新增编辑页 | 类型：业务单据
> 本页用于 WMS 手动创建 TR 和编辑草稿 TR；字段定义以 `调拨单字段清单.md` 为准。

## 1. 页面定位

调拨单新增编辑页面向仓库主管，用于手动创建仓间调拨需求，并在草稿态维护调出仓、调入仓、调拨类型、商品明细和调出数量。页面只处理 `source_type=wms_manual` 的调拨单；ERP 下发调拨由接口生成，本页不提供人工新建入口。

草稿态保存不变更库存；点击确认调出后才触发调出仓扣减现存、调拨在途增加，并进入已调出（在途）状态。已调出或已调入的 TR 不允许再进入本页编辑。

## 2. 页面结构

```text
调拨单新增编辑页
├─ 顶部摘要
│  ├─ 返回列表
│  ├─ 页面标题：新建调拨单 / 编辑调拨单
│  └─ 调拨单号、状态（编辑态展示）
├─ 基本信息区
│  ├─ 调拨类型
│  ├─ 调出仓、调入仓
│  ├─ 商品种数、调拨总数
│  └─ 备注
├─ 商品明细区
│  ├─ 新增行
│  ├─ 商品编码、商品名称、规格、单位
│  ├─ 调出仓当前现存/可用
│  ├─ 调出数量
│  └─ 移除行
└─ 底部动作区
   ├─ 返回
   ├─ 保存草稿
   └─ 确认调出
```

## 3. 页面模式

| 模式 | 进入方式 | 页面表现 | 主要动作 |
|:--|:--|:--|:--|
| 新增 | 列表页点击新建调拨单 | 空表单，来源默认为 WMS 手动，状态为草稿 | 返回、保存草稿、确认调出 |
| 编辑草稿 | 草稿 TR 点击编辑 | 调拨单号和状态只读，基础信息和明细可编辑 | 保存草稿、确认调出 |
| 非草稿访问 | 已调出或已调入 TR 访问编辑路由 | 不进入编辑态，跳转详情页或执行页 | 查看详情、进入执行页 |

## 4. 控件说明

| 控件 | 说明 |
|:--|:--|
| 调拨类型 | 必选；手动新增默认 `warehouse_transfer` 仓间调拨；`erp_transfer` 由 ERP 接口下发，本页不创建 ERP 来源 TR |
| 调出仓 | 必选，仅可选择启用仓库；切换调出仓后清空商品明细并重新加载该仓库存商品 |
| 调入仓 | 必选，仅可选择启用仓库，不能等于调出仓 |
| 商品种数 | 只读，按明细行中不重复 SKU 数量统计 |
| 调拨总数 | 只读，按 `Σ line_transfer_out_qty` 实时汇总 |
| 备注 | 可选，用于记录调拨用途、交接说明或异常备注，最长 200 字符 |
| 新增行 | 从调出仓有库存的商品中新增一行；无可用商品时提示不可新增 |
| 商品编码 | 必选，选择后带出商品名称、规格、单位和调出仓库存数量 |
| 调出数量 | 必填，正整数，必须大于 0 且不超过调出仓可用数量 |
| 移除行 | 仅从当前草稿明细中移除该行，不影响商品主数据和库存 |

## 5. 核心交互

| 交互 | 说明 |
|:--|:--|
| 新建调拨单 | 从列表页进入，页面生成空草稿表单，未保存前不产生 TR 单号 |
| 保存草稿 | 校验调拨类型、调出仓、调入仓、商品明细和调出数量，成功后生成或更新 TR 草稿 |
| 编辑草稿 | 仅状态=草稿时允许编辑；保存后仍保持草稿，不变更库存 |
| 切换调出仓 | 清空已选商品明细，避免跨仓库存口径混用 |
| 新增商品行 | 商品来源于调出仓当前库存；同一 SKU 不允许重复加入 |
| 确认调出 | 先保存当前表单，再执行确认调出；成功后状态变为已调出（在途）并跳转详情页 |

## 6. 校验与反馈

| 场景 | 反馈 |
|:--|:--|
| 未选择调拨类型 | 阻断保存/确认，提示“请选择调拨类型” |
| 未选择调出仓 | 阻断保存/确认，提示“请选择调出仓” |
| 未选择调入仓 | 阻断保存/确认，提示“请选择调入仓” |
| 调出仓=调入仓 | 阻断保存/确认，提示“调出仓和调入仓不能相同” |
| 未添加商品明细 | 阻断保存/确认，提示“请至少添加一条商品明细” |
| 商品重复 | 阻断保存/确认，提示“该商品已在明细中存在” |
| 调出数量为空或≤0 | 阻断保存/确认，提示“调出数量必须大于 0” |
| 调出数量大于可用数量 | 阻断确认调出，提示“调出仓可用库存不足” |
| 备注超长 | 阻断保存/确认，提示“备注不能超过 200 字符” |
| 非草稿单据进入编辑 | 跳转详情页，不允许修改调拨头和调出数量 |

## 9. 不展示内容

- 不展示 ERP 下发调拨的新建入口；ERP 来源 TR 由接口生成后进入执行页处理。
- 不展示调入清点、确认调入、差异入库和 BL 生成表单；这些动作在执行页完成。
- 不展示审核、反审核、删除调拨单操作；状态变更必须通过动作按钮触发。
- 不展示第三方物流、承运商、运输轨迹等信息。
