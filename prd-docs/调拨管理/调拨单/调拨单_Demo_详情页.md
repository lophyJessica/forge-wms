# 调拨单_Demo_详情页

> 角色：Demo·详情页 | 类型：执行作业单
> 详情页用于查看 TR 来源、状态、库存影响、差异、关联 BL 和操作记录。

## 1. 页面定位

调拨单详情页面向研发、测试、产品复核，用于确认 TR 是否按“草稿 → 已调出（在途） → 已调入（完成）”流转，库存是否按调出/调入规则过账，差异是否正确触发 BL。

## 2. 页面结构

```text
调拨单详情
├─ 顶部摘要
│  ├─ 调拨单号、状态、是否差异
│  └─ 返回列表、进入执行页、查看BL
├─ 调拨头信息
│  ├─ 调拨类型、来源、来源单号
│  ├─ 调出仓、调入仓
│  └─ 调出总数、实收总数、差异总数、在途总数
├─ 调拨明细
│  ├─ 商品、规格、单位
│  ├─ 调出数量、实收数量、差异数量、明细状态
│  └─ 调出货位、调入货位、差异原因
├─ 库存影响
│  ├─ 调出确认前后快照
│  └─ 调入确认前后快照
├─ 关联单据
│  ├─ ERP 来源单号
│  ├─ 库存流水 FL
│  └─ 报损单 BL
└─ 操作记录
   └─ 创建、保存、确认调出、登记实收、确认调入、生成BL
```

## 3. 展示规则

| 区域 | 展示规则 |
|:--|:--|
| 顶部摘要 | 状态只展示 TR 自身状态：草稿、已调出（在途）、已调入（完成） |
| 来源信息 | 来源=ERP 下发时展示来源单号；WMS 手动时来源单号展示 `-` |
| 调拨明细 | 数量右对齐；差异数量>0 的行标记差异 |
| 库存影响 | 展示现存、占用、可用、在途；差异场景补充差异待核销口径 |
| 关联 BL | 仅当 `blNo` 有值时展示入口；BL 套件已产出，本套件仅定义 TR 差异触发/关联口径，详见 报损管理/报损单/ |
| 操作记录 | 所有状态变更必须有操作人和操作时间 |

## 4. 动作规则

| 动作 | 展示条件 | 说明 |
|:--|:--|:--|
| 进入执行页 | 状态=草稿或已调出（在途） | 跳转执行页处理确认调出或调入 |
| 查看 BL | 状态=已调入（完成）且有关联 BL | 跳转 BL 详情；BL 套件已产出，本套件仅定义 TR 差异触发/关联口径，详见 报损管理/报损单/ |
| 查看库存流水 | 有 FL 记录 | 跳转库存流水列表并带来源单号筛选 |
| 返回列表 | 全部状态 | 返回调拨单列表 |

> 详情页不展示新增、审核、反审核、删除操作。

## 8. 测试关注点

- TR 详情能追溯 WMS 手动或 ERP 下发来源。
- TR 状态不出现待审核、已审核、反审核等审核流状态。
- 已调出（在途）状态下在途数量大于 0，已调入（完成）后在途归零。
- 正常调拨不生成 BL。
- 到货差异生成 BL，原因固定为调拨损耗。
- 库存影响区展示现存、占用、可用、在途；差异场景补充差异待核销口径。
- BL 套件已产出，本套件仅定义 TR 差异触发/关联口径，详见 报损管理/报损单/。
