# 调拨单_Demo_列表页

> 角色：Demo·列表页 | 类型：执行作业单
> 页面只描述结构、控件、交互和 Mock；字段定义以 `调拨单字段清单.md` 为准，不重复维护字段枚举。

## 1. 页面定位

调拨单列表页用于 PC 端查看 TR 的创建、调出、在途、调入完成和差异情况。列表页承担查询和入口分发，不直接编辑状态；状态变更进入执行页后通过动作按钮完成。

## 2. 页面入口

| 入口 | 说明 |
|:--|:--|
| 菜单 | 调拨管理 / 调拨单 |
| 关联跳转 | ERP 调拨来源、库存查询、库存流水、报损单 BL |

## 3. 页面结构

```text
调拨单列表
├─ 顶部操作区
│  └─ 新建调拨单
├─ 筛选区
│  ├─ 关键词：调拨单号/来源单号/SKU/商品名称
│  ├─ 状态、调拨类型、来源、是否差异
│  ├─ 调出仓、调入仓
│  └─ 创建时间范围、查询、重置
├─ 表格区
│  ├─ 调拨单号、状态、调拨类型、来源、来源单号
│  ├─ 调出仓、调入仓、调出总数、实收总数、差异总数、在途总数
│  ├─ 关联BL、创建时间、调出时间、调入时间
│  └─ 行操作：执行、查看详情、查看BL
└─ 分页
   └─ 默认 20 条/页
```

## 4. 行操作规则

| 场景 | 行操作 |
|:--|:--|
| 状态=草稿 | 执行、查看详情 |
| 状态=已调出（在途） | 执行、查看详情 |
| 状态=已调入（完成）且无差异 | 查看详情 |
| 状态=已调入（完成）且有差异 | 查看详情、查看 BL |
| 来源=ERP 下发 | 展示来源单号；来源字段只读 |

> 列表页不提供“直接改状态”。确认调出、登记实收、确认调入均在执行页完成。

## 5. 查询规则

- 关键词支持调拨单号、ERP 来源单号、商品编码、商品名称模糊查询。
- 状态筛选只使用 TR 自身状态：草稿、已调出（在途）、已调入（完成）。
- 是否差异根据 `hasDifference` 判断。
- 调出仓和调入仓可单独筛选，支持查询某仓待调出或待收货任务。
- 分页默认 20 条/页。

## 7. 空状态与异常态

| 状态 | 展示 |
|:--|:--|
| 无数据 | `暂无调拨单` |
| 查询无结果 | `未找到符合条件的调拨单` |
| 接口失败 | 保留筛选条件，提示 `调拨单列表加载失败，请重试` |
| ERP 同步失败 | 行内展示同步失败标识，可进入详情查看操作记录 |
