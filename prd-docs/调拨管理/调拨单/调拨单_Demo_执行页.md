# 调拨单_Demo_执行页

> 角色：Demo·执行页 | 类型：执行作业单
> 本页用于调拨单创建、调出确认、调入清点和差异入库动作；字段定义以 `调拨单字段清单.md` 为准。

## 1. 页面定位

调拨单执行页面向仓库主管、调出库作业员和调入库作业员。页面按 TR 状态切换操作区：草稿阶段处理调拨需求和确认调出，已调出（在途）阶段处理调入清点和确认调入，已调入（完成）阶段只读展示结果。

## 2. 页面结构

```text
调拨单执行页
├─ 顶部摘要
│  ├─ 调拨单号、状态、调拨类型、来源
│  └─ 返回列表、查看详情
├─ 调拨头信息
│  ├─ 调出仓、调入仓、来源单号
│  └─ 计划调出日期、计划调入日期、备注
├─ 明细作业区
│  ├─ 商品、规格、单位
│  ├─ 调出数量、实收数量、差异数量、在途数量
│  └─ 调出货位、调入货位、差异原因
├─ 库存提示区
│  ├─ 调出仓现存/占用/可用
│  └─ 调入仓现存/占用/可用/在途
├─ 差异区
│  ├─ 差异总数
│  └─ 预计生成BL提示
└─ 底部动作区
   ├─ 保存草稿
   ├─ 确认调出
   ├─ 登记实收
   ├─ 确认调入
   └─ 差异入库
```

## 3. 页面模式

| TR 状态 | 页面表现 | 主要动作 |
|:--|:--|:--|
| 草稿 | 调拨头和调出数量可编辑，ERP 来源字段只读 | 保存草稿、确认调出 |
| 已调出（在途） | 调出字段只读，实收数量可编辑 | 登记实收、确认调入、差异入库 |
| 已调入（完成） | 全部字段只读，展示库存影响和关联 BL | 查看详情、查看 BL |

## 4. 核心交互

| 交互 | 说明 |
|:--|:--|
| 新建调拨单 | 从列表页进入草稿模式，仅适用于 WMS 手动来源 |
| ERP 下发执行 | 从 ERP 来源 TR 进入，来源单号和下发明细只读 |
| 保存草稿 | 校验调出仓、调入仓、商品、调出数量，不变更库存 |
| 确认调出 | 校验调出仓可用库存，成功后状态变为已调出（在途） |
| 登记实收 | 调入库按明细录入实收数量，允许分批保存但不变更状态 |
| 确认调入 | 所有实收=调出时展示，完成正常入库 |
| 差异入库 | 存在实收<调出时展示，提示将生成 BL 后完成 |

## 5. 校验与反馈

| 场景 | 反馈 |
|:--|:--|
| 调出仓=调入仓 | 阻断保存，提示“调出仓和调入仓不能相同” |
| 调出数量为空或≤0 | 阻断保存/确认，提示“调出数量必须大于 0” |
| 调出仓可用不足 | 阻断确认调出，提示“调出仓可用库存不足” |
| 已调出后修改调出数量 | 不允许修改，字段只读 |
| 实收数量大于调出数量 | 阻断确认，提示“实收数量不能大于调出数量” |
| 差异入库 | 二次确认文案包含“将生成报损单 BL，原因：调拨损耗” |
| BL 生成失败 | 阻断完成，提示“差异报损单生成失败，请重试” |

## 8. 不展示内容

- 不展示审核、反审核动作。
- 不展示删除入口；调拨单状态通过动作按钮推进。
- 不展开 BL 新增、审核、详情字段；只展示 TR 关联 BL 的跳转入口。
- 不展示运输轨迹、快递单、承运商交接等第三方物流信息。
