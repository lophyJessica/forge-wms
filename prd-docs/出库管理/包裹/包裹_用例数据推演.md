## 6. 用例 4：库存占用不足回滚

### 输入

| 字段 | 值 |
|:--|:--|
| 包裹号 | PKG20260705-000004 |
| SKU | SKU002 晨光按动式中性笔黑色 |
| 出库数量 | 6 |
| 过账前现存 | 80 |
| 过账前占用 | 5 |
| 过账前可用 | 75 |

### 校验

| 校验 | 公式 | 结果 |
|:--|:--|:--|
| 现存充足 | `80 >= 6` | 通过 |
| 占用充足 | `5 >= 6` | 不通过 |

### 期望结果

- 库存校验失败。
- PKG 保持包装中，不变为已包装。
- 现存、占用、可用均不变。
- 不生成 FL，页面展示过账失败原因。

## 9. 三口径对账解释

| 阶段 | 现存 | 占用 | 可用 | 解释 |
|:--|--:|--:|--:|:--|
| SO 审核前 | 120 | 0 | 120 | 商品尚未被销售订单锁定 |
| SO 审核后/包装前 | 120 | 10 | 110 | 订单占用 10 件，销售可用已减少 |
| 包装完成后 | 110 | 0 | 110 | 仓库实物出库 10 件，同时释放订单占用 |
| 交运确认后 | 110 | 0 | 110 | 交运不再改变库存 |

## 10. 走查清单

| 检查项 | 预期 |
|:--|:--|
| PKG 前缀 | 已在 `09-单据命名规则` 定义，6 位序号 |
| FL 前缀 | 已在 `09-单据命名规则` 定义，8 位序号 |
| 来源限制 | 只能由 CHECK 已复核下推，不提供新增入口 |
| 状态 | 仅待包装、包装中、已包装、已作废 |
| 包装资料 | 重量、承运商、面单、打印、贴单均需满足 |
| 库存过账 | 包装完成时扣减现存、释放占用，可用保持不变 |
| FL | 包装完成时生成一条 FL |
| 交运 | 交运不再触发库存过账 |
