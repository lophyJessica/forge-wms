## 6. 用例 4：少交阻断

### 输入

| 字段 | 值 |
|:--|:--|
| 应交包裹数 | 2 |
| 实扫包裹数 | 1 |
| 未扫包裹 | PKG20260705-000002 |

### 校验

| 校验 | 公式 | 结果 |
|:--|:--|:--|
| 数量一致 | `actual_pkg_count = should_pkg_count` | `1 = 2` 不成立 |

### 期望结果

- 阻断确认交运。
- 提示 `存在未交接包裹，不能确认交运`。
- DSH 保持交运中。
- 不回传 TMS/ERP/财务，不完结订单。

## 9. 走查清单

| 检查项 | 预期 |
|:--|:--|
| DSH 前缀 | 已在 `09-单据命名规则` 定义 |
| 来源限制 | 只能由已包装 PKG 聚合生成，不提供新增入口 |
| 状态 | 仅待交运、交运中、已交运、已作废 |
| 包裹校验 | 包裹必须属于当前 DSH 且已包装 |
| 数量校验 | 应交数=实交数才允许确认 |
| 回传 | 确认交运后回传 TMS/ERP/财务结果 |
| 订单完结 | 交运确认后订单已交运/完结 |
| 库存过账 | 交运不做任何库存过账，不生成出库 FL |
