# 库存查询_Demo_列表页

> 状态：已补全
> 角色：Demo·列表页 | 类型：查询页
> 页面定位：库存查询只读列表页，包含筛选区、表头、Mock 数据。
> 字段定义：以《库存查询字段清单》为准，本文件只描述页面呈现。

## 1. 页面概述

| 项目 | 内容 |
|:--|:--|
| 页面名称 | 库存查询 |
| 页面路径 | `/inventory/stock-query` |
| 页面类型 | 查询页 |
| 页面结构 | 筛选区 + 列表区 + 分页 |
| 访问端 | PC/PDA |
| 禁止入口 | 新增、编辑、详情、删除、调整库存、状态 Tab |

## 2. 筛选区

### 2.1 PC 筛选区

| 控件 | 类型 | 占位/展示 | 说明 |
|:--|:--|:--|:--|
| 仓库 | SelectWithSearch | 全部授权仓库 | 单选 |
| 库区 | SelectWithSearch | 全部库区 | 受仓库过滤 |
| 货位 | ScanInput/Input | 扫描或输入货位 | 支持货位编码/条码 |
| 商品 | SelectWithSearch/Input | 商品编码/名称/条码 | 支持扫码枪输入 |
| 批次 | Input | 输入批次 | 模糊匹配 |
| 不显示零库存 | Switch | 默认关闭 | 开启后隐藏 `现存 <= 0` |
| 搜索 | Button | 搜索 | 执行查询 |
| 重置 | Button | 重置 | 清空筛选条件 |

### 2.2 PDA 筛选区

| 控件 | 类型 | 说明 |
|:--|:--|:--|
| 仓库 | Select | 默认当前作业仓 |
| 扫货位 | ScanInput | 扫描货位条码后自动带入 |
| 扫商品 | ScanInput | 可选，扫描商品条码进一步过滤 |
| 批次 | Input | 可选输入 |
| 查询 | Button | 大按钮，适合手套操作 |

## 3. 表头

三口径 `现存 / 占用 / 可用` 必须同屏相邻展示；`冻结 / 在途` 紧跟其后展示。

| 列名 | 字段编码 | PC 展示 | PDA 展示 | 说明 |
|:--|:--|:--:|:--:|:--|
| 仓库 | warehouseName | 是 | 否 | PDA 可在顶部摘要展示 |
| 库区 | zoneName | 是 | 否 | - |
| 货位 | locationCode | 是 | 是 | PDA 主信息 |
| SKU | productCode | 是 | 是 | 商品编码 |
| 商品 | productName | 是 | 是 | 商品名称 |
| 规格 | productSpec | 是 | 否 | 无值展示 `-` |
| 单位 | unit | 是 | 是 | - |
| 批次 | batchNo | 是 | 是 | 无批次展示 `-` |
| 现存 | qtyOnHand | 是 | 是 | 数量右对齐 |
| 占用 | qtyAllocated | 是 | 是 | 数量右对齐 |
| 可用 | qtyAvailable | 是 | 是 | 数量右对齐，公式校验 |
| 冻结 | qtyFrozen | 是 | 是 | 数量右对齐 |
| 在途 | qtyInTransit | 是 | 是 | 数量右对齐 |
| 最近流水 | lastFlNo | 是 | 否 | `FL` 单号 |
| 最近变动时间 | lastChangedAt | 是 | 否 | `YYYY-MM-DD HH:mm:ss` |

列表不设置操作列。

## 5. 交互状态

| 状态 | 展示 |
|:--|:--|
| 首次进入 | 默认按授权范围查询第 1 页，20 条/页 |
| 查询中 | 表格骨架屏，不清空筛选条件 |
| 查询成功 | 展示列表、分页、查询时间 |
| 空结果 | 展示“当前筛选条件下暂无库存记录” |
| 查询失败 | Toast：`查询失败，请稍后重试` |
| 无权限 | Toast：`无可查看仓库权限` |

## 6. 页面约束

- 不展示状态 Tab。
- 不展示新增、编辑、详情、删除、调整库存入口。
- 不提供冻结、解冻、调拨确认等动作按钮。
- 不展示导出按钮；如后续范围变更需增加导出，应单独补充权限与验收。
