# 库存查询主PRD

> 状态：已补全
> 角色：主PRD | 类型：查询页
> 读者：研发工程师、测试工程师、产品复核
> 权威层级：context/ > 本文件 > Demo 文件

## 1. 业务背景

库存查询是 Forge WMS 库存管理模块的即时库存查询页，用于让仓管员、仓库主管、产品/测试复核人员在 PC 或 PDA 上按仓库、库区、货位、商品、批次多维度查看当前库存。

本页面解决三个核心问题：

- 仓库现场需要快速确认某个货位、某个商品、某个批次当前还有多少实物库存。
- 出库、盘点、调拨等流程复核时，需要区分库存是否已被销售占用、是否冻结、是否在途。
- 产品和测试需要基于统一口径校验库存数据：`可用 = 现存 - 占用 - 冻结`。

## 2. 功能范围

### 2.1 In Scope

| 范围 | 说明 |
|:--|:--|
| 多维度库存查询 | 支持按仓库、库区、货位、商品、批次组合筛选 |
| PC/PDA 访问 | PC 以表格为主；PDA 支持扫码货位/商品后查看精简列表 |
| 三口径同屏 | 现存、占用、可用必须在同一列表中同时展示 |
| 状态量展示 | 同行展示冻结、在途，帮助判断不可用原因 |
| 零库存过滤 | 支持隐藏 `现存 <= 0` 的记录 |
| 只读追溯 | 展示最近库存流水 `FL` 与最近变动时间，用于排查数据来源 |

### 2.2 Out Scope

| 范围 | 说明 |
|:--|:--|
| 库存调整 | 本页不提供调整库存入口；调整需走盘点/差异调整等业务流程 |
| 新增/编辑/删除库存 | 库存由入库、出库、盘点、调拨、报损等动作产生，不允许在查询页维护 |
| 业务单据详情页 | 本套件仅覆盖查询列表页，不新增详情页 |
| 状态审批流 | 查询页无草稿、待审核、已确认等业务状态 |
| 成本核算 | 不展示库存成本、金额、账龄等财务口径 |

## 3. 单据定位

| 项目 | 内容 |
|:--|:--|
| 页面定位 | 即时库存只读查询页 |
| 单据属性 | 不属于业务单据，不生成库存查询单号 |
| 数据粒度 | 默认按 `warehouse_code + zone_code + location_code + sku_code + batch_no` 聚合拆行 |
| 数据来源 | 库存余额聚合结果 + 库存流水 `FL` + 仓库/库区/货位/商品主数据快照 |
| 上游影响 | 入库上架、包装出库、盘点调整、调拨确认、报损等动作改变库存 |
| 下游用途 | 为出库占用校验、PDA 找货、库存盘点前核对、产品/测试复核提供只读依据 |

## 4. 业务场景

| 场景ID | 场景 | 用户 | 说明 |
|:--|:--|:--|:--|
| S01 | 按仓库/库区查库存 | 仓库主管 | 查看上海一仓存储区当前各货位库存分布 |
| S02 | 按货位查库存 | 仓管员 | PDA 扫描 `LOC-A0102`，查看该货位有哪些 SKU 和批次 |
| S03 | 按商品/批次查库存 | 仓管员/测试 | 输入 `SKU003` + 批次 `B20260705-A`，核对不同货位的现存、占用、可用 |
| S04 | 查冻结库存 | 仓库主管 | 查看质检中、盘点中或报损触发的冻结数量，不在查询页解冻 |
| S05 | 查在途库存 | 仓库主管 | 查看调拨途中数量，用于确认调入库尚未收货的库存 |
| S06 | 空结果核对 | 测试 | 输入不存在的批次，页面展示空状态，数量汇总为 0 |

## 5. 状态机

不适用(查询页无语态流转)。

原因：库存查询是只读台账视图，不承载业务单据生命周期，也不提供新增、编辑、审核、确认、作废等动作。库存数量的变化必须由入库、出库、盘点、调拨、报损等业务动作触发，并按 `context/06-库存管理规则.md` 生成或更新库存流水 `FL`。本页面仅读取结果，不改变库存状态。

## 6. 规则摘要

| 规则ID | 规则 | 说明 |
|:--|:--|:--|
| INVQ-R01 | 查询页只读 | 不提供新增、编辑、删除、审核、调整库存入口 |
| INVQ-R02 | 三口径公式 | `可用 = 现存 - 占用 - 冻结` |
| INVQ-R03 | 现存公式 | `现存 = 期初 + Σ入库 - Σ出库 + Σ盘盈 - Σ盘亏` |
| INVQ-R04 | 在途独立展示 | 在途来源于调拨途中，不参与可用公式 |
| INVQ-R05 | 明细粒度 | 默认按仓库、库区、货位、商品、批次拆行展示 |
| INVQ-R06 | 权限过滤 | 先按用户可见仓库范围过滤，再执行用户输入筛选条件 |
| INVQ-R07 | 零库存过滤 | 开启后隐藏 `现存 <= 0` 的行 |
| INVQ-R08 | 分页 | PC 默认每页 20 条；PDA 可使用同一查询接口按移动端分页展示 |
| INVQ-R09 | 空值展示 | 空批次、空最近流水等字段展示 `-` |

## 7. 字段清单摘要

字段定义以《库存查询字段清单》为 SSOT。本主 PRD 只列字段分组，不重复定义字段枚举。

| 分组 | 代表字段 |
|:--|:--|
| 查询条件 | 仓库、库区、货位、商品、批次、不显示零库存 |
| 列表展示 | 仓库、库区、货位、商品、批次、现存、占用、可用、冻结、在途、最近流水 |
| 系统字段 | 查询时间、分页参数、权限范围、记录唯一键 |

## 8. 验收标准

| 验收项 | 输入/前置 | 预期结果 |
|:--|:--|:--|
| AC01 三口径公式 | 某行现存 120、占用 30、冻结 10 | 可用展示 80 |
| AC02 冻结展示 | 某行冻结 12 | 同行冻结列展示 12，可用按公式扣减 |
| AC03 在途展示 | 调拨单 `TR20260706-0001` 调出确认、调入未确认 | 在途列展示对应数量，不参与可用公式 |
| AC04 多维筛选 | 仓库 + 库区 + 货位 + SKU + 批次组合筛选 | 仅返回匹配明细行 |
| AC05 零库存过滤 | 开启不显示零库存 | `现存 <= 0` 的行不展示 |
| AC06 分页 | 授权范围内返回 45 行 | 默认第 1 页 20 行，第 2 页 20 行，第 3 页 5 行 |
| AC07 空结果 | 输入不存在批次 `NOT-EXIST-2026` | 展示空状态，汇总数量为 0 |
| AC08 只读约束 | 打开库存查询页 | 无新增、编辑、详情、删除、调整库存入口 |
| AC09 PDA 查询 | PDA 扫描货位条码后查询 | 返回该货位库存明细，字段口径与 PC 一致 |
