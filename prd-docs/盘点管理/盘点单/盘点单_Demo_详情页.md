# 盘点单_Demo_详情页

> 角色：Demo·详情页 | 类型：执行作业单
> 详情页用于查看盘点单范围、冻结状态、实盘明细、差异报告、主管审核、调整结果和关联 FL。

## 1. 页面定位

盘点单详情页面向研发、测试、产品复核提供完整核对视图。它需要证明 CK 覆盖了盘点全流程：创建、锁定范围、PDA 实盘、差异报告、主管审核、库存调整和 FL 追溯。

## 2. 页面结构

```text
盘点单详情
├─ 顶部摘要
│  ├─ 盘点单号、状态、盘点类型、差异标记
│  └─ 返回列表、开始盘点、进入PDA、生成差异报告、主管重盘、审核通过并确认调整、查看FL
├─ 盘点头信息
│  ├─ 仓库、盘点范围、盘点人、审核人
│  └─ 创建时间、锁定时间、提交审核时间、调整时间
├─ 冻结范围
│  └─ 货位、冻结状态、禁止出入库提示
├─ 盘点明细
│  ├─ 货位、商品、系统数、实盘数、差异数、差异率、差异类型
│  └─ 明细状态、实盘人、实盘时间
├─ 差异报告
│  ├─ 差异行汇总、最大差异率、是否触发重盘
│  └─ 盘盈/盘亏调整预览
├─ 关联 FL
│  └─ FL 单号、变动类型、变动数量、变动后现存
└─ 操作记录
   └─ 创建、开始盘点、扫码、生成报告、重盘、调整
```

## 3. 展示规则

| 区域 | 展示规则 |
|:--|:--|
| 顶部摘要 | 状态只展示 CK 自身状态：草稿、盘点中、待审核、已调整 |
| 盘点头信息 | 进入盘点中后范围、盘点人、审核人只读 |
| 冻结范围 | 盘点中展示“货位冻结，禁止出入库”；已调整展示“盘点冻结已解除” |
| 盘点明细 | 数量右对齐；差异数正数展示盘盈，负数展示盘亏 |
| 差异报告 | 仅待审核和已调整状态展示；差异率 >5% 的行标记需重盘 |
| 关联 FL | 已调整后展示；草稿、盘点中、待审核展示 `-` |
| 库存说明 | 展示“确认调整才更新现存并生成 FL；占用不变，可用按公式重算” |

## 4. 动作规则

| 动作 | 展示条件 | 说明 |
|:--|:--|:--|
| 开始盘点 | 状态=草稿 | 锁定范围，货位冻结，状态进入盘点中 |
| 进入 PDA | 状态=盘点中 | 打开 PDA 实盘作业入口 |
| 生成差异报告 | 状态=盘点中且全部明细已盘 | 状态进入待审核 |
| 主管重盘 | 状态=待审核且存在差异率 >5% 或主管判定需复核 | 状态回到盘点中 |
| 审核通过并确认调整 | 状态=待审核且无超阈值未重盘明细 | 更新现存，生成 FL，状态进入已调整 |
| 查看 FL | 状态=已调整且有关联 FL | 跳转库存流水 |
| 返回列表 | 全部状态 | 返回盘点单列表 |

> 详情页不展示删除、反审核、独立差异调整单等入口。

## 7. 测试关注点

- 详情页能展示完整状态链路和操作日志。
- 待审核状态必须能看到主管重盘和审核确认调整动作。
- 差异率 >5% 时不能直接确认调整。
- 已调整状态能追溯 FL，FL 来源单号为当前 CK。
- 页面明确展示盘点调整只影响现存，盘盈 `+N`、盘亏 `-N`，占用不变。
